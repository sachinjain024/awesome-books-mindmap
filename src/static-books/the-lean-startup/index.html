The Lean Startup

How Today's Entrepreneurs Use Continuous Innovation to Create Radically Successful Businesses by Eric Ries

About This Mind Map

The Lean Startup revolutionized how companies are built and new products are launched. Eric Ries introduces a methodology that applies scientific experimentation to entrepreneurship. This mind map presents the core Build-Measure-Learn framework and the principles for creating a successful startup through validated learning, rapid iteration, and customer feedback.

Core Principles of The Lean Startup

  • Build-Measure-Learn — The fundamental feedback loop that powers all startup progress. Build an MVP, measure customer response, learn from the data, and iterate.
  • Validated Learning — Progress in a startup is measured by validated learning about customers, not by creating features or hitting milestones.
  • Minimum Viable Product — The fastest way to get through the Build-Measure-Learn loop. Start with the simplest version that lets you start learning.
  • Innovation Accounting — A new way to measure progress when traditional metrics fail. Focus on actionable metrics, not vanity metrics.
  • Pivot or Persevere — Regularly evaluate whether to change direction (pivot) or stay the course. A structured change designed to test a new hypothesis.
  • Small Batches — Working in small batches to reduce cycle time and get feedback faster. Continuous deployment enables rapid iteration.
  • Engines of Growth — Three sustainable engines: sticky (retention), viral (word-of-mouth), and paid (customer acquisition). Focus on one at a time.
  • Five Whys — Ask "why" five times to get to the root cause of problems. Make proportional investments to fix issues at their source.
