## ğŸš€ Quick Start

Want to add your own book mind map? It's easy!

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Basic knowledge of Markdown

### Installation

```bash
# Clone the repository
git clone https://github.com/sachinjain024/awesome-books-mindmap.git

# Navigate to the project directory
cd awesome-books-mindmap

# Install dependencies
npm install
```

### Generate a Mind Map

Simply run the build command with your book request:

```bash
# Generate a new mind map
npm run build
```

Then, in your LLM (Claude, ChatGPT, etc.), use this prompt:

```
Generate a mindmap for <Book Name> by <Author Name>
```

**Example:**
```
Generate a mindmap for "Atomic Habits" by "James Clear"
```

The AI will:
1. âœ… Research the book structure
2. âœ… Create markdown files with frontmatter
3. âœ… Generate chapter summaries
4. âœ… Add book-specific styling
5. âœ… Update the homepage

Then build the site:

```bash
npm run build
```

Your new mind map will be available at `docs/<book-name>/index.html`!
